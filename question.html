Question-1: What are the different ways to select an element in DOM?

There is many ways to select element in DOM : 

                        1.getElementById:
                        example:const element = document.getElementById('Id');
                        2.getElementsByClassName:
                        example:const elements = document.getElementsByClassName('Class');
                        3.getElementsByTagName:
                        example:const elements = document.getElementsByTagName('div/p/h/span/button all tag can be here');
                                
                        4.querySelector:
                        example:const elements = const element = document.querySelector('.Class'); 
                        const element = document.querySelector('#Id');
                        5.querySelectorAll:
                        example:const elements = const element = document.querySelectorAll('.Class'); 
                        const element = document.querySelectorAll('#Id');
These are the main 5 way to select element in DOM. There are other way to do that but this 5 way we use mostly.

Question-2: What is difference between innerHtml,innerText and textContent?

Difference between innerHtml,innerText and innerContent are written down below:
1.innerhtml: It actually indicate Html content of an element. example: 

                                  <div id="maindiv" class="p-2">
                                  <h1>< class="font-bold text-xl">Question-1:></h1>
                                  </div>
                                  <script>
                                     console.log(document.getElementById("maindiv").innerHtml);
                                     //Output: "<h1> class="font-bold text-xl">Question-1:</h1> "
                                  </script>
2.innerText: It indicate the inner all visiable text content of an element. example: 
                                     <div id="maindiv" class="p-2">
                                       <h1>< class="font-bold text-xl">Question-1:></h1>
                                     </div>
                                     <script>
                                        console.log(document.getElementById("maindiv").innerText);
                                        //Output: "Question-1:"
                                     </script>
3. textContent:It indicate the inner all visiable and hidden text content of an element also all children nodes. example: 

                                     <div id="maindiv" class="p-2">
                                     <h1>< class="font-bold text-xl">Question-1:></h1>
                                     <p> I am Newaz</p>
                                     </div>
                                     <script>
                                        console.log(document.getElementById("maindiv").innerText);
                                        //Output: "Question-1: I am Newaz"
                                    </script>

Question-3: What is the event delegation in the DOM?

Event Delegation is a technique to add event listener in multiple child.
html code: 

                                <div>
                                    <button> class="btn">Button 1</button>
                                    <button> class="btn">Button 1</button>
                                    <button> class="btn">Button 1</button>
                                </div>

Js code: 
                                 document.querySelectorAll(".btn").forEach(button => {
                                        button.addEventListener("click", () => { 
                                            console.log("Button clicked!");
                                        });
                                    });

Question-4: What is event bubbling in the DOM?

When a parent and children both have event listener then a kind problem happen when clicked that element that is called Event Bubbling in the DOM.
html code: 

                                <div id="parent">
                                    <button id="child">Click Me </button>
                                </div>
js code: 

                                <script>
                                document.getElementById("child").addEventListener("click", function () {
                                    console.log("Button Clicked!");
                                });
    
                                document.getElementById("parent").addEventListener("click", function () {
                                    console.log("Parent Div Clicked!");
                                });
    
                                document.body.addEventListener("click", function () {
                                    console.log("Body Clicked!");
                                });
                            </script>
We can prevent it by using event.stopPropagation().

Question-5: How do you create, add, and remove elements using JavaScript?

There is several way tocreate add and remove element in javascript.
Example:
Create element: 

                             const main_heading= document.createElement('h1'); 
                             main_heading.textContent='Hello World!';
Add element: Let's assume there is a div which id is "main-content". 
                            <div id="main-content"> 

                            </div>
Js code: 
                        const main_heading= document.createElement('h1');
                        main_heading.textContent='Hello World!';
                        const main_box=document.getElementById('main-content');
                        main_box.appendChild(main_heading);
Remove element:
Html Code: 
                            <div id="main-content">
                            <h1 id="main-heading">Hello World!</h1>
                            </div>
Js code: 
                            const main_box=document.getElementById('main-content');
                            const child=document.getElementById('main-heading');
                            main_box.removeChild(child);
This is who we can create , add and remove eliment by Js.
